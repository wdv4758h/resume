Build
========================================

Install Dependencies
------------------------------

Arch Linux:

.. code-block:: sh

    sudo pacman -S texlive-bin texlive-core texlive-fontsextra texlive-latexextra

    # or install the meta package "texlive-most"


Generate
------------------------------

.. code-block:: sh

    make
